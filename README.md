# Bitrix24 MCP Server

MCP-сервер для интеграции Claude с Битрикс24.

## Возможности

- **CRM**: лиды, сделки, контакты (чтение, создание, обновление)
- **Задачи**: список, создание, обновление, завершение
- **Телефония**: статистика звонков, история
- **Пользователи**: список сотрудников, профили

---

## Деплой на Railway (пошагово)

### 1. Загрузить код на GitHub
1. Создайте новый репозиторий на https://github.com/new
2. Загрузите все файлы из этой папки:
   - `server.py`
   - `requirements.txt`
   - `Procfile`
   - `railway.toml`

### 2. Создать проект на Railway
1. Зайдите на https://railway.app и войдите через GitHub
2. Нажмите **New Project → Deploy from GitHub repo**
3. Выберите ваш репозиторий
4. Railway автоматически определит Python и установит зависимости

### 3. Добавить переменную окружения
1. В проекте Railway → вкладка **Variables**
2. Добавьте переменную:
   ```
   BITRIX_WEBHOOK = https://verona24.bitrix24.ru/rest/18451/k60gw5gxlgaolxvo
   ```

### 4. Получить публичный URL
1. Вкладка **Settings → Networking → Generate Domain**
2. Railway выдаст URL вида: `https://your-app.up.railway.app`

---

## Подключение к Claude.ai

1. Откройте https://claude.ai
2. Зайдите в **Settings → Integrations**
3. Нажмите **Add custom integration**
4. Вставьте URL вашего Railway сервера:
   ```
   https://your-app.up.railway.app/sse
   ```
5. Сохраните — интеграция появится в чате
